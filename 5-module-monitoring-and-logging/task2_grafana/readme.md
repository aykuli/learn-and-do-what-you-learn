## Задание 2

**Panels**:
1) утилизация CPU для nodeexporter (в процентах, 100-idle):

![](./assets/2.png)

```promql
(1 - sum(rate(node_cpu_seconds_total{mode="idle"}[5m])) / sum(rate(node_cpu_seconds_total[5m])))*100
```

В формуле обозначения:
* `rate(node_cpu_seconds_total{mode="idle"}[5m])` — высчитывает, сколько секунд в минуту (в интервале  5 минут) процессор простаивал.
* `sum(...)` — складывает показатели простоя по всем ядрам процессора вместе - у меня их на личном ПК 20.
* `sum(rate(node_cpu_seconds_total[5m]))` — время простоя на общее время всех ядер.

* После деления получается доля свободного времени (например, 0.75).
* `1 - (доля свободного времени)` — получаем долю занятого времени.
* умножаю на 100 для получения знаечния в виде процентов.

2) CPULA 1/5/15;
